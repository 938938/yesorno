## 사용방법
1. 제목 아래의 입력창에 묻고 싶은 질문을 입력해주세요!
2. 입력창을 클릭했을 때 나오는 예시 질문을 사용하셔도 좋습니다.
3. 예시 질문은 키보드의 화살표 키로도 선택할 수 있습니다.
4. 키보드의 Enter 키 혹은 묻기! 버튼을 눌러 랜덤 대답을 받아보세요!
